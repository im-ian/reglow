# Reglow + Preact example

A runnable “Northstar Launch Room” settings flow using typed Reglow Custom Elements directly in
Preact JSX.

```sh
pnpm --filter @reglow/example-preact dev
```

The example demonstrates complex `options` property forwarding, controlled `value` / `checked`
state, native `input` events, and the typed `onrg-press` custom-event prop.
